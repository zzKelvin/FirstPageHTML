  • Funcionamento de uma Lista

    • Descrição: No HTML, as listas são usadas para apresentar informações em uma sequência ordenada ou desordenada.
    Existem dois tipos principais de listas: listas ordenadas (ol) e listas desordenadas (ul).

    Listas Ordenadas (ol): São usadas para listas em que a ordem dos itens é importante. Cada item da lista é marcado com um (li) e numerado automaticamente.

    Listas Desordenadas (ul): São usadas para listas onde a ordem dos itens não é importante. Cada item da lista também é marcado com um (li), mas em vez de números, os itens geralmente são marcados com pontos ou outro marcador.

  • Funcionamento de um titulo

    • Descrição: Os títulos (como (h1), (h2), etc.) são usados para estruturar e organizar o conteúdo de uma página da web, fornecendo um esquema hierárquico. Eles são importantes tanto para os usuários quanto para os motores de busca, pois destacam os tópicos principais e sub-tópicos da página.

    Hierarquia: Títulos vão de (h1) a (h6), com (h1) sendo o mais importante ou de maior nível. Geralmente, há apenas um (h1) por página (o título principal), seguido por (h2) para subseções e assim por diante.

    Estilo: Visualmente, os títulos são geralmente maiores e mais destacados do que o texto normal, ajudando na leitura e navegação da página.

  • Funcionamento de um paragrafo

    • Descrição: O elemento de parágrafo (p) é usado para agrupar sentenças relacionadas formando um bloco de texto. Ele serve para organizar o conteúdo em unidades lógicas e de fácil leitura.

    Estrutura: Cada parágrafo é usualmente iniciado em uma nova linha com um espaço entre ele e outros blocos de texto para melhorar a legibilidade.

    Fluxo do Conteúdo: Os parágrafos ajudam a apresentar as ideias de forma clara e ordenada, permitindo que o leitor acompanhe o argumento ou a narrativa de forma coerente.
